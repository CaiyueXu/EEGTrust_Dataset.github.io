EEGTrust Dataset
A dataset for human-robot trust research with EEG

Abstract

The EEGTrust dataset provides a comprehensive collection of neurophysiological signals recorded during human-robot collaborative interactions in a controlled gaming environment. This dataset features electroencephalography (EEG) data from 16 university students who participated in a cooperative Overcooked-AI game with robot agents.

The experimental design systematically manipulated robot capability and task difficulty to induce different levels of trust, which were captured through both subjective post-trial ratings with 5-point Likert scale and continuous EEG measurements using a 64-channel system. Each participant completed 30 trials with 60 seconds each. Additionally, face video was recorded to provide multimodal insights into trust dynamics.

This resource enables researchers to investigate the neural correlates of human-robot trust, develop trust detection algorithms, and advance understanding of the cognitive processes underlying collaborative human-robot interactions.

How to Use

If you are interested in using this dataset, you will have to print, sign and scan an EULA (End User License Agreement) and send it via email to the dataset maintenance team. We will then send you a download link and a password to download the data through email. Please head on over to the downloads page for more details.

Also, please consult the dataset description page for a complete explanation of the dataset.
